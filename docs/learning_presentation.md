# 学習管理プラットフォーム「Learning」
## 高校音楽教育向け 導入説明資料

---

## 1. このプラットフォームでできること（一言で）

> **先生が生徒の練習進捗をリアルタイムで管理・把握でき、LINEで自動通知まで完結する、音楽部・音楽科向けの学習管理システムです。**

---

## 2. 現場の課題

| よくある課題 | このプラットフォームでの解決 |
|-------------|--------------------------|
| 生徒ごとの練習状況が把握しにくい | ダッシュボードで全生徒の進捗を一覧表示 |
| 課題を配布しても提出確認が大変 | 課題配布・完了管理を一元化。未提出者には自動リマインダー |
| 個別フィードバックに時間がかかる | 達成記号（⭐/△/×）とコメントで素早く評価 |
| 生徒への連絡手段がバラバラ | LINE連携で通知・リマインダーを自動送信 |
| バンド全体の練習管理が難しい | バンドグループ機能で合奏練習も一元管理 |
| 進捗データが蓄積・分析できない | 月間レポート・CSV出力で客観的データを取得 |

## 4. 先生向け機能

### 4-1. 生徒管理

- 生徒を登録し、**パート（Vocal / Guitar / Bass / Drums / Keyboard）** や学年で分類
- **学校グループ（クラス）** 単位でまとめて管理
- ステータス（在籍中 / 休会 / 卒業）で状態を追跡

### 4-2. トレーニング管理

- 「基礎」→「安定」→「応用」→「実践」の**4段階レベル**でトレーニングを定義
- 各トレーニングに**期間・達成基準・週間目標・先生コメント**を設定
- 生徒ごとに割り当てて、進捗を達成記号で管理
  - ⭐ = 達成
  - △ = 進行中
  - × = 未着手

### 4-3. バンド練習管理

- バンドグループを作成して複数生徒を割り当て
- **合奏練習のタスク**も個人練習と同様に定義・追跡
- 関連パートを指定してアンサンブル練習を管理

### 4-4. 課題配布

- タイトル・説明・期限付きの課題を**複数生徒に一括配布**
- 課題配布時に**LINE自動通知**
- 未提出の生徒へ**LINEリマインダーをワンクリック送信**
- 完了率をダッシュボードでリアルタイム確認

### 4-5. 先生向けダッシュボード

- **全生徒の進捗を一覧表示**（週間成長指標・課題完了率・通知候補）
- グループ別・パート別にフィルタして確認
- 進捗データを**CSVでエクスポート**（保護者報告・学校への報告に活用）

### 4-6. 自動通知・LINE連携

- 生徒がLINEと連携すると、以下の通知を**自動送信**：
  - 課題配布通知
  - 未提出リマインダー
  - 週間サマリー
  - 長期間ログイン・練習がない生徒への再活性化メッセージ
- **テンプレート管理**でメッセージを事前に作成・使い回し
- 通知送信前に**プレビュー確認**が可能

## 5. 生徒向け機能（学習ポータル）

生徒は**専用URLをLINEやメールで受け取り**、ログイン不要で学習ポータルにアクセスできます。

| 機能 | 内容 |
|------|------|
| トレーニング一覧 | 先生に割り当てられた練習課題と進捗を確認 |
| 練習ログ記録 | いつ・何を練習したかを日単位で記録 |
| 課題確認 | 締切・内容・完了状態を確認 |
| 努力ポイント | 練習実績に応じてポイントが貯まる |
| 学習ストリーク | 連続ログイン日数が表示され、継続を促進 |
| クラス内ランキング | グループ内で努力ポイントランキングを表示 |

> **ゲーム感覚の要素（ポイント・ストリーク・ランキング）で、生徒のモチベーションを自然に維持します。**

## 6. LINE連携のしくみ

```
① 先生が生徒のLINE連携QRコードを発行（24時間有効）
        ↓
② 生徒がQRコードを読み込み、LINEで友だち追加
        ↓
③ 連携完了！以降は自動で通知が届く

【送られる通知の例】
  📚 「今週の課題が届きました：ドレミの音階練習」
  ⏰ 「提出期限まで3日です！忘れずに。」
  🎉 「今週の練習サマリー：5回ログイン、⭐3つ達成」
```

## 7. 導入の流れ

```
STEP 1　申し込み
  └─ 学校名・担当者名・生徒数を入力してお申し込み

STEP 2　初期設定（先生）
  └─ 先生がアカウント作成 → 学校グループ設定 → 生徒を登録

STEP 3　生徒への案内
  └─ 先生がポータルURLをメール送信 → 生徒がLINE連携

STEP 4　日常利用開始
  └─ 先生：ダッシュボードで管理、課題配布、LINE通知
     生徒：ポータルで進捗確認、練習ログ記録
```

## 8. 管理職・学校側へのメリット

| メリット | 詳細 |
|---------|------|
| 教員の業務効率化 | 進捗確認・連絡業務がシステムで自動化され、指導時間に集中できる |

## 10. よくある質問

**Q. 生徒がスマホを持っていない場合は使えますか？**
A. ポータルはPCブラウザでも利用できます。LINE連携はオプションのため、連携なしでも基本機能は利用可能です。

**Q. クラスが複数ある場合はどうなりますか？**
A. 「学校グループ」機能で複数クラスに分類して管理できます。クラスごとにダッシュボードを切り替えて確認できます。

**Q. 先生が複数いる場合は共有できますか？**
A. 同一学校内で複数の先生アカウントを作成し、生徒情報を共有して使用できます。

**Q. 導入後のサポートはありますか？**
A. 導入後のガイドページと、使い方サポートをご用意しています。
