# Singing Empty State Copy Guidelines

Singing の空状態 UI は、ユーザーに「何もない」と感じさせる場所ではなく、次の一歩を見つけられる場所として設計する。

このガイドは Encouragement Inbox / Community Feed / Journey / Circle Discovery など、Singing 内で空状態コピーや CTA を追加・調整するときの軽量な判断基準として使う。

Singing Home のカード単位の表示順、CTA、Service Object 方針は `docs/handover/singing_home_card_guidelines.md` も参照する。

## 基本方針

空状態は、欠けているものを強調しない。

避けたい見え方:

```text
何もありません
```

目指す見え方:

```text
次の一歩が見える場所
```

Singing の空状態では、未達成・不足・人気・順位ではなく、音楽を続けるきっかけ、仲間とのつながり、挑戦の入口を伝える。

## NG 表現

以下のような表現は、寂しさ、失敗、比較、人気のなさを強めやすいので避ける。

- まだ誰にも応援されていません
- 投稿がありません
- データがありません
- 人気がありません
- ランキング外です
- 失敗しています

## OK 表現

以下のような方向性で、前向きで軽い一歩を案内する。

- まずは一歩
- 自分のペースで
- 音楽の輪
- 仲間とのつながり
- 挑戦を重ねる
- 続けていく

コピーは「診断しないと進めない」ではなく、「診断や挑戦をきっかけに音楽の輪が広がる」という見え方に寄せる。

## CTA 設計

空状態には、可能なら 1〜2 個の CTA を置く。

CTA は選択肢を増やしすぎず、その画面で自然に次へ進めるものを優先する。

- 今日の診断をする
- Community Feedを見る
- プロフィールを整える
- 同じCircleの仲間を見る
- チャレンジを始める

CTA の優先順位は、画面の目的に合わせる。

- 診断起点の画面: 「今日の診断をする」
- コミュニティ起点の画面: 「Community Feedを見る」または「同じCircleの仲間を見る」
- プロフィール起点の画面: 「プロフィールを整える」
- 挑戦起点の画面: 「チャレンジを始める」

## セクション別方針

### Encouragement Inbox

「応援がない」ことを伝えるのではなく、これからつながりが生まれる入口として見せる。

推奨コピー:

```text
あなたの一歩が、仲間とのつながりになります。
```

CTA 例:

- 今日の診断をする
- プロフィールを整える

### Community Feed

「投稿がない」ことを伝えるのではなく、自分の診断や挑戦から音楽の輪が広がることを伝える。

推奨コピー:

```text
まずはあなたの診断から、音楽の輪を広げていきましょう。
```

CTA 例:

- 今日の診断をする
- チャレンジを始める

### Circle Discovery

「該当者がいない」ことを伝えるのではなく、少しずつ仲間と出会っていく場所として見せる。

推奨コピー:

```text
少しずつ音楽の輪を広げていきましょう。
```

CTA 例:

- 同じCircleの仲間を見る
- Community Feedを見る

### Music Journey

診断なしの場合は、基本的に Journey セクション自体を非表示にする。

どうしても空状態が必要な場合だけ、音楽の旅の入口として短く案内する。

推奨コピー:

```text
ここから音楽の旅をはじめましょう。
```

CTA 例:

- 今日の診断をする

## 実装時チェック

- 「ありません」「いません」「失敗」「ランキング外」だけで終わっていないか
- 人気、順位、評価の低さを連想させていないか
- ユーザーの次の一歩が 1 つ以上見えているか
- CTA が画面の目的と合っているか
- 診断、挑戦、仲間、音楽の輪のいずれかにつながっているか
- 空状態が目立ちすぎて、通常状態より重い体験になっていないか

## 今後参照するフェーズ

このガイドは、Singing 内で以下を追加・調整するフェーズで参照する。

- Encouragement Inbox の空状態コピーや CTA
- Community Feed / Growth Feed の初期状態
- Circle Discovery / 近い仲間表示の該当なし状態
- Challenge Circle の参加前・活動前状態
- Music Journey / Growth Journey の診断なし状態
- Singing TOP でコミュニティ導線を追加する画面

ゴールは、Singing の空状態を「寂しい空白」ではなく「次の一歩を案内する場所」として統一すること。
