学習管理プラットフォーム「Learning」

先生の管理業務を効率化し、生徒のモチベーションを高める音楽教育システム
このプラットフォームでできること
先生が生徒の練習進捗をリアルタイムで管理・把握でき、LINEで自動通知まで完結する、音楽部・音楽科向けの学習管理システムです。
現場の課題と解決策
よくある課題 このプラットフォームでの解決
生徒ごとの練習状況が把握しにくい ダッシュボードで全生徒の進捗を一覧表示
課題を配布しても提出確認が大変 課題配布・完了管理を一元化。未提出者には自動リマインダー
個別フィードバックに時間がかかる 達成記号（⭐ / △ / ×）とコメントで素早く評価
生徒への連絡手段がバラバラ LINE連携で通知・リマインダーを自動送信
バンド全体の練習管理が難しい バンドグループ機能で合奏練習も一元管理
進捗データが蓄積・分析できない 月間レポート・CSV出力で客観的データを取得
先生向け主要機能
📋 生徒管理
  • パート・学年・クラスで分類
  • ステータス管理（在籍中 / 休会 / 卒業）
  • 複数クラスを学校グループで整理
🎯 トレーニング管理
  • 4段階レベル（基礎→安定→応用→実践）
  • 達成基準・週間目標・期間を設定
  • 達成記号（⭐/△/×）で素早く評価
🎸 バンド練習管理
  • バンドグループを作成・生徒を割り当て
  • 合奏練習タスクを定義・追跡
  • 関連パートを指定して管理
📝 課題配布
  • 複数生徒へ一括配布
  • 配布時にLINE自動通知
  • 完了率をリアルタイム確認
📊 ダッシュボード・分析
  • 全生徒の進捗を一覧表示
  • 週間成長指標・課題完了率
  • CSV出力（報告・保護者対応に活用）
🔔 自動通知管理
  • リマインダー・週間サマリーを自動送信
  • テンプレートを事前作成して使い回し
  • 送信前にプレビューで内容確認

よくある質問
生徒がスマホを持っていない場合は使えますか？
ポータルはPCブラウザでも利用できます。LINE連携はオプションのため、連携なしでも基本機能は利用可能です。
クラスが複数ある場合はどうなりますか？
「学校グループ」機能で複数クラスに分類できます。クラスごとにダッシュボードを切り替えて確認できます。
先生が複数いる場合は共有できますか？
同一学校内で複数の先生アカウントを作成し、生徒情報を共有して使用できます。
導入後のサポートはありますか？
導入後のガイドページと使い方サポートをご用意しています。
